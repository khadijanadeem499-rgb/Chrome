**🌐 Chrome Profiles Manager**

A clean, interactive profile selection interface that lets users manage multiple Chrome profiles — perfect for separating work, personal, and family browsing experiences.

**✨ Features**

**Profile Cards** — Display each Chrome profile with avatar, name, badge, and sync status

**Edit Name & Picture** — Click the pencil icon to edit profile name or overlay to change avatar

**Add New Profile** — Quick "Add" card with tooltip to create Work, Personal, or School profiles

**Sync Indicator** — Animated icon showing active sync status

**Guest Mode** — Quick access to guest browsing

**Show on Startup** — Checkbox preference with informative tooltip

**Responsive Layout**— Cards adjust gracefully across devices

**🛠️ Technologies**

- HTML5
  
- CSS3 (External: css/style.css)
  
- JavaScript (included via external file)
  
- Font Awesome Icons
  
- Custom avatar images
